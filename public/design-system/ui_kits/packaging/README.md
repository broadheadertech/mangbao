# Packaging — UI Kit

Print-and-pack artboards for Mang Guabao's physical products. These are the brand applied to the things you actually hold.

## What's in here

```
index.html               — all artboards laid out on a canvas
BaoWrapper.jsx           — square wrapper with corner blob accents
DrinkCup.jsx             — yellow Calamansi Lemonade cup
TrayLiner.jsx            — Bao White cream stripe liner
TakeoutBag.jsx           — kraft-style paper bag with stamped wordmark
ReceiptCard.jsx          — "Sulit na sulit ka dito" receipt thank-you
NeonStorefront.jsx       — in-store neon sign
```

## Print specs (suggested, flag with brand owner)
* Bao wrapper: 14" × 14" greaseproof paper, food-safe ink (1-color Saging or 2-color Saging + Saffron).
* Drink cup: 16oz double-walled, wrap-around 1-color print on Saffron base.
* Tray liner: 13" × 9" parchment, 1-color Saging on Bao White.
* Bag: kraft #4 lunch bag, stamped wordmark.
